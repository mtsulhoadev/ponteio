# ⛽ Ponteio — Sistema de Fechamento de Caixa

Sistema web em PHP para gerenciamento do fechamento de caixa diário dos postos de combustível da Rede Ponteio.

## Funcionalidades

- **Login por funcionário** com controle de perfil (frentista e gerente)
- **Dashboard** com resumo do dia e ações rápidas
- **Mapa de Caixa digital** fiel ao formulário físico, com todos os meios de pagamento da rede (Elo, Visa, Master, Amex, Diners, VR Auto, Shell Box, Sorocred, Ticket, AME e outros)
- **Cálculo automático** em tempo real de totais, subtotais e diferença de caixa
- Estrutura preparada para **30 postos** e múltiplos funcionários

## Stack

- PHP 7.4+
- MySQL / MariaDB
- Apache (XAMPP para desenvolvimento local)
- HTML/CSS/JS puro (sem frameworks externos)
